\# Changelog



\## \[1.0.0] - 2026-02-05



\### Added

\- 实现 Config 模块，支持 YAML 配置文件解析与验证

\- 实现 FileFilter 模块，支持按文件大小过滤（含 Mock 测试）

\- 实现 Renamer 模块，支持正则批量重命名与序号添加

\- 支持 dry\_run 预览模式，防止误操作

\- 10 个自动化测试（单元测试 + Mock 测试 + 集成测试）

\- GitHub Actions CI/CD 流水线，自动跑测试

\- 测试覆盖率 96%

\- CLI 命令行工具，支持终端直接调用

\- 支持 pip 本地安装
